# Modern Auth Backend (Open Source)

A robust and scalable authentication system built with **TypeScript**, **Drizzle ORM**, and **PostgreSQL**. This project is designed to promote open-source collaboration, streamline the development process, and serve as a high-quality learning resource.

---

## Purpose
The core objectives of this project are:
1. **Open Source:** Building high-quality tools in collaboration with the community.
2. **Efficiency:** Saving developers' time by providing a ready-to-use authentication boilerplate.
3. **Continuous Learning:** Exploring and implementing modern tech stacks like Drizzle and PostgreSQL.

## Tech Stack
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Package Manager:** [pnpm](https://pnpm.io/)
* **ORM:** [Drizzle ORM](https://orm.drizzle.team/)
* **Database:** [PostgreSQL](https://www.postgresql.org/)
* **Code Quality:** [ESLint](https://eslint.org/) (Ensuring clean and maintainable code)

## Roadmap & Features
Our primary focus is on building a secure and complete authentication flow:
- [ ] **User Registration:** Secure signup with hashed passwords.
- [ ] **Login:** JWT-based user authentication.
- [ ] **Email Verification:** OTP-based account activation logic.
- [ ] **Resend OTP:** Handling expired or lost verification tokens.
- [ ] **Session Management:** Secure token refresh strategies.
